## Project Overview
PayZenith is a web-based Payroll Management System designed to simplify employee management, attendance tracking, and salary processing.

## Features
- User-friendly dashboard
- Employee management
- Attendance tracking
- Payroll calculation
- Responsive UI

## Technology Stack
- Frontend: React + Vite
- Styling: Tailwind CSS
- Backend: Cloud-based backend services
- Database: Managed database service

## How to Run the Project
1. Install dependencies
   npm install
2. Start the development server
   npm run dev
3. Open the application in the browser
